Part-1: Implement K-mean from Scratch (50 marks)¶

Some of the brief description of algorithm is as follows:

  1. Place K centroids at random locations.
  2. Assign all the data points to the closest centroid.
  3. Compute the new centroids as the mean of all points in the cluster
  4. Compute the Sum of Squared errors between new and old centroids

Part 2-SKLEARN K-Mean using Elbow Method (30 marks)¶

Now we will use employee dataset to find clusters from it using the elbow method. In a K-means clustering, the elbow approach is a graphical representation of selecting the best "K." It works by finding WCSS (Within-Cluster Sum of Square) i.e. the sum of the square distance between points in a cluster and the cluster centroid.

In this part, You must draw an elbow graph where the values of the WCSS (on the y-axis) correspond to the various values of K (on the x-axis). We choose the K-value at which the elbow is generated when we notice an elbow shape on the graph.This area is known as the elbow point. Beyond the Elbow point, increasing the value of ‘K’ does not lead to a significant reduction in WCSS. Therefore, we would select the optimal cluster value "K" at the elbow point.

Q: By changing the eplison and minimum sample parameters, what could be the effect on result? ( 1 mark )

ANS: The epsilon value determines the radius around each point within which the algorithm searches for other points to form a cluster. Increasing the epsilon value will increase the size of the neighborhoods and result in fewer clusters. Decreasing the epsilon value will decrease the size of the neighborhoods and result in more clusters whereas The minimum samples value determines the minimum number of points required to form a dense region or a cluster. Increasing the minimum samples value will result in fewer clusters because it requires more points to form a cluster. Decreasing the minimum samples value will result in more clusters because it requires fewer points to form a cluster

Q: Why do we need DBScan when we have K-Mean Clustering? What are some of its advantages? (2 marks)

ANS: 1)K-means can only find clusters that are spherical and have a similar size. DBSCAN, on the other hand, can find clusters of any shape and size.

2) K-means assigns every point to a cluster, even if it doesn't belong to any cluster. DBSCAN can identify noise points that don't belong to any cluster.

3) K-means requires us to specify the number of clusters beforehand. DBSCAN does not require this information and can determine the number of clusters automatically
